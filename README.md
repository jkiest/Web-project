# 榮紹精密塑膠射出 - 前端展示網頁

## 📝 專案簡介
本專案是為「榮紹精密塑膠射出股份有限公司」開發的企業官方展示網頁。採用響應式設計（Responsive Web Design），提供工廠內況展示、部門介紹以及聯絡資訊等功能。

## 🛠️ 技術棧
- **前端框架**: Bootstrap 5.1.3 (負責佈局與響應式導覽列)
- **動態效果**: Anime.js (標題文字動態載入)
- **交互功能**: Typewriter-effect (打字機效果)、Scrollspy (捲動監控)
- **圖標庫**: Font Awesome

## ✨ 核心功能
- **動態標題**: 使用 Anime.js 實作字母由透明轉顯示的進場特效。
- **部門資訊導覽**: 整合卡片式佈局展示「塑膠射出部」、「射出模具部門」等六大部門資訊。
- **互動式模態框**: 讓客戶能透過內建表單提交聯繫資訊。
- **響應式佈局**: 針對不同螢幕尺寸優化，確保在行動裝置與桌面端皆有良好體驗。
